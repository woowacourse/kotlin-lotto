- [x] 구매 금액 입력 기능 구현

- [x] 당첨 번호 입력 기능 구현
- [x] 보너스 번호 입력 기능 구현
- [x] 당첨 통계 출력 기능 구현

- 메서드 체이닝 단점 분석
- assertJ 메서드 체이닝 활용
- CsvSource로 변경 가능한 테스트 수정
- 총 수익률 계산 표본 관련 고민 및 수정
